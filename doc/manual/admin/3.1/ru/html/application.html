Аннотация

Для разширения функциональности OTRS-фреймворка, можно также установить дополнительные пакеты приложений. Это можно сделать с помощью менеджера пакетов из Панели Администрирования, который загружает приложения из онлайн репозитория и управляет зависимостями между этими пакетами. Также приложения можно установить из локальных файлов.

FAQ (Часто Задаваемые Вопросы)

FAQ - это компонент Базы Знаний. Он позволяет редактировать и просматривать статьи. Статьи можно просматривать и ограничивать для агентов, клиентов и пользователей или анонимных пользователей. Также они могут быть структурированы по группам и быть написаны на различных языках.
