前言

本书是为OTRS系统管理员准备的，也是OTRS新手的一本好参考书。

下面的章节描述了OTRS软件的安装、配置和管理。本书的前三分之一用来描述OTRS的主要功能，余下的部分是所有可配置参数的参考。

本书仍然是一个进展中的工作,正往新版本前进中。我们需要你的反馈以编制一本高质量的参考文档--可用、精确且完整的文档。如果你发现本书有内容缺失，或没有解释得足够清楚的事情，甚至发现了拼写错误、语法错误、错别字，都请写给我们。通过我们的BUG跟踪系统 http://bugs.otrs.org 提交任何反馈均高度赞赏。提前感谢你的贡献 ！
