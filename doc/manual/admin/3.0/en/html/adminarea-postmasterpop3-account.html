Using mail accounts

There are several possibilities to transport new emails into the ticket system. One of them is the otrs.PostMaster.pl module that pipes the mails directly into the system. Another possibility is the use of mail accounts which can be administrated through the web interface. The "PostMaster Mail Accounts" link on the Admin page loads the management console for the mail accounts (see Figure 5.48 below). OTRS supports the mail protocols: POP3, POP3S, IMAP and IMAPS.

Figure 5.48. Mail account management.
