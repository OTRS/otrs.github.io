Admin email

OTRS administrators can send messages to specific users or groups. The "Admin Notification" link on the Admin page opens the screen where the agents and groups that should be notified can be selected (see Figure 5.53 below).

Figure 5.53. Admin notification.

It is possible to specify the sender, subject and body text of the notification. You can also select the agents, groups and roles who should receive the message.
