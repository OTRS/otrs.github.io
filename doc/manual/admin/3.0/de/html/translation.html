Kapitel 15. Übersetzung des OTRS-Frontends

OTRS bietet Mehrsprachigkeit in der Weboberfläche an.

Wie der Lokalisationsmechanismus des OTRS Frameworks arbeitet und welche Möglichkeiten er bietet, entnehmen Siie bitte dem Kapitel Language Translations aus dem Developer Handbuch auf http://doc.otrs.org. Dieses Kapitel beschreibt, wie eigene Übersetzungen eingepflegt und komplett neue Sprachen hinzugefügt werden können.
