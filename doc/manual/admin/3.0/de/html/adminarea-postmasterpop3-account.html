Einrichten von Mail-Konten

Es gibt mehrere Möglichkeiten, Mails in OTRS einzuspeisen. Neben der Nutzung des PostMaster.pl-Skripts, welches die Mails direkt in das System piped, können auch Mail-Konten abgefragt werden, an die Mails für Ihr Ticket-System gehen. Über den Link "PostMaster Mail Account" innerhalb des Admin-Bereiches erreichen Sie die Verwaltung der Mail-Konten, die OTRS automatisch auf neue Nachrichten überprüfen soll. OTRS ünterstützt dabei die Mail-Protokolle POP3, POP3S, IMAP und IMAPS.
