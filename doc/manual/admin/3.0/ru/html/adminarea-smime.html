SMIME

OTRS может обрабатывать входящие S/MIME-кодированные сообщения и подписывать исходящие сообщения электронной почты. Перед тем как использовать эту возможность, ее необходимо активировать и настроить путем изменения некоторых параметров конфигурации в файле SysConfig.

Ссылка "S/MIME Certificates" на Панели Управления OTRS позволяет управлять вашими S/MIME сертификатами (см. ниже Рисунок 5.44). Можно добавлять или удалять сертификаты а также производить поиск в SMIME-данных.
