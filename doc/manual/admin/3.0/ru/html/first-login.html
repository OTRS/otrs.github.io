Первое применение

Перейдите к странице входа, как это описано в разделе Агентский веб-интерфейс . Введите имя пользователя и пароль. Ввиду того что после новой установки в системе нету ни одного зарегистрированного пользователя, войдите с правами OTRS-администратора, используя для этого имя пользователя 'root@localhost' и пароль 'root'.

Внимание

Данные доступа этого аккаунта действительны для каждой копии новоустановленной системы OTRS. Именно по этой причине пароль OTRS-администратора необходимо изменить как можно быстрее. Это можно сделать после входа в систему с правами администратора перейдя по ссылке preferences.

Если не хотите входить в систему с правами OTRS-администратора, просто введите имя и пароль своего обычного агентского аккаунта.

В случае потери пароля всегда можно отправить запрос в систему для его восстановления. Для этого просто нажмите на ссылку, которая находится под кнопкой Вход, введите адрес своей электронной почты, который вы указали в настройках своего OTRS-аккаунта и нажмите кнопку Отправить (см. ниже Рисунок 4.4).
