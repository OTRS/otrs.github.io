Эллектронная почта администратора

OTRS администраторы могут отсылать сообщения определенному пользователю или группе. Переход по ссылке "Уведомления Администратора" в Панели Администрирования открывает страницу, на которой можно указать агентов и группы, которые должны быть оповещены (см. ниже Рисунок 5.53).

Рисунок 5.53. Оповещения Администратором.

Можно указать отправителя, тему и текст уведомления. Также можно выбрать агентов, группы и роли, которые должны получить сообщение.
