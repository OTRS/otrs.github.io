Системный журнал

Перейдя по ссылке "Системный журнал" в Панели Администрирования можно просмотреть записи в системном журнале, отсортированные в обратном хронологическом порядке (см. ниже Рисунок 5.56).

Рисунок 5.56. Системный журнал.

Каждая строка в журнале содержит: время, приоритет, компонент системы и непосредственно саму запись.

Замечание

Системный журнал доступен только через веб-интерфейс на Linux/UNIX системах. В Windows-системах, журнал можно увидеть, октрыв в текстовом редакторе файл [install_dir]otrs\var\log\otrs.log.
