Состояния

Используя ссылку "Состояния" на Панели Управления OTRS, можно управлять различнми статусами заявок, которые вы хотите использовать в системе (см. ниже Рисунок 5.46).

Рисунок 5.46. Управление состоянимями.

После установки в системе будут следующие предустановленные состояния:

  • закрыта успешно

  • закрыта неуспешно

  • обьедененные

  • новая

  • открытая

  • ожидание авто закрытие+

  • ожидание авто закрытие-

  • ожидание напоминание

  • удаленная

Каждое состояние связано с типом, который нужно указать когда создается новое состояние. По умолчанию есть такие типы состояний:

  • закрытая

  • обьедененные

  • новая

  • открытая

  • ожидание авто

  • ожидание напоминание

  • удаленная
