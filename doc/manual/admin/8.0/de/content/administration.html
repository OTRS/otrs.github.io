Administration¶

Jedes System benötigt eine Konfiguration. Die Konfiguration eines Systems sollte eine einfache Aufgabe und die Konfigurationstools für den Zweck geeignet sein.

OTRS bietet mehrere Administrationstools zum Konfigurieren, Überwachen, Steuern und Erweitern von OTRS.
