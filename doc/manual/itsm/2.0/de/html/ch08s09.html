OTRS::ITSM beinhaltet die nachstehenden Reports zur Ermittlung wichtiger Kennzahlen für die Steuerung des Change Managements. Alle Reports können über den integrierten Report-Generator aufgerufen werden.

Anzahl Changes innerhalb einer definierten Zeitspanne

Der Report erlaubt die Definition einer relativen (Changes innerhalb der letzten x-Tage) oder absoluten Zeitspanne (Datum 1 - Datum 2). Zusätzlich besteht die Möglichkeit die zu reporteten Change-Status zu definieren.

Als Ausgabeformat stellt der Report "CSV" oder "Print" (PDF) zur Verfügung.
