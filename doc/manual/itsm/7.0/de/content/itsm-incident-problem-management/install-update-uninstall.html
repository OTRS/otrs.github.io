Installieren, Aktualisieren und Deinstallieren¶

Dieses Paket kann mit der Paketverwaltung im Administrator-Interface oder mit der Kommandozeile installiert, aktualisiert oder deinstalliert werden.

Bemerkung

Sie müssen in der Gruppe Admin sein, um auf das Agenten-Interface zugreifen und die Paketverwaltung https://doc.otrs.com/doc/manual/admin/7.0/de/content/administration/package-manager.html nutzen zu können. Das ist im Administrator-Handbuch detaillierter beschrieben.

Bemerkung

Wir nehmen an, dass der Benutzername otrs für OTRS erstellt wurde und es in /opt/otrs installiert wurde. Wenn Sie andere Benutzer oder Installationspfade haben, müssen Sie das Kommando zur Installation entsprechend ändern. Führen Sie dieses Kommando in der Kommandozeile aus:

otrs> /opt/otrs/bin/otrs.Console.pl Search 'Admin::Package'

Führen Sie dann das benötigte Kommando mit --help aus, um die Möglichkeiten zu sehen.

Anforderungen¶

  • OTRS Framework 7.0.x
  • Allgemeiner Katalog 7.0.x
  • ITSM Core 7.0.x

Installieren¶

Installieren via Administrator-Interface¶

So installieren Sie das Paket mit der Paketverwaltung:

  1. Öffnen Sie die Paketverwaltung im Administrator-Interface.
  2. Klicken Sie auf die Schaltfläche Durchsuchen… in der linken Seitenleiste.
  3. Wählen Sie eine .opm Datei aus Ihrem lokalen Dateisystem aus.
  4. Klicken Sie auf die Schaltfläche Paket installieren.
  5. Befolgen Sie die Installationsanleitung.
  6. Nach der Installation wird das Paket im Widget Lokale Verzeichnisse dargestellt.

Installation via Kommandozeile¶

So installieren Sie das Paket über die Kommandozeile:

  1. Speichern Sie die opm Datei in einem Verzeichnis, in dem der OTRS-Benutzer Leserechte hat.

  2. Führen Sie dieses Kommando aus:

    otrs> /opt/otrs/bin/otrs.Console.pl Admin::Package::Install /path/to/Package-x.y.z.opm

Aktualisieren¶

Aktualisieren via Administrator-Interface¶

So aktualisieren Sie dieses Paket mit der Paketverwaltung:

  1. Öffnen Sie die Paketverwaltung im Administrator-Interface.
  2. Klicken Sie auf die Schaltfläche Durchsuchen… in der linken Seitenleiste.
  3. Wählen Sie eine .opm Datei, die neuer als das installierte Paket ist.
  4. Klicken Sie auf die Schaltfläche Paket installieren.
  5. Befolgen Sie die Anweisungen zur Aktualisierung.
  6. Nach dem Aktualisieren, wird das Paket im Widget Lokales Verzeichnis angezeigt.

Aktualisieren via Kommandozeile¶

So aktualisieren Sie das Paket über die Kommandozeile:

  1. Speichern Sie die opm Datei in einem Verzeichnis, in dem der OTRS-Benutzer Leserechte hat.

  2. Führen Sie dieses Kommando aus:

    otrs> /opt/otrs/bin/otrs.Console.pl Admin::Package::Upgrade /path/to/Package-x.y.z.opm

Deinstallieren¶

Warnung

Wenn Sie dieses Paket deinstallieren, werden die Ticket-Typen und Status, die bei der Installation hinzugefügt wurden, deaktiviert. Sie können im Administrator-Interface wieder aktiviert werden.

Die Statistiken, die mit diesem Paket verknüpft sind, sind nicht mehr verfügbar, wenn das Paket deinstalliert wird.

Deinstallieren via Administrator-Interface¶

So deinstallieren Sie dieses Paket mit der Paketverwaltung:

  1. Öffnen Sie die Paketverwaltung im Administrator-Interface.
  2. Wählen Sie das Paket im Widget Lokales Verzeichnis.
  3. Klicken Sie auf den Deinstallieren-Link in der Spalte Aktion.
  4. Befolgen Sie die Deinstallationsanleitung.

Deinstallieren via Kommandozeile¶

So deinstallieren Sie dieses Paket über die Kommandozeile:

  1. Führen Sie dieses Kommando aus:

    otrs> /opt/otrs/bin/otrs.Console.pl Admin::Package::Uninstall PACKAGE_NAME
