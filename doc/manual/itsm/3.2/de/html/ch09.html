Kapitel 9. Release Management

Der Release Management Prozess wird in einer späteren Version von OTRS ITSM implementiert sein. Die grundlegenden Informationen lassen sich jedoch bereits ab der Version 1.0 konfigurieren, erfassen und steuern.

Es können z. B. Genehmigungs-Regeln oder Übersichten der DSL (Definitive Software Library) konfiguriert und genutzt werden.
