Chapter 9. Release Management

Please note that the we are planning to introduce the release management process with a future OTRS ITSM version. Basic information, however, can be configured, captured and controlled as of version 1.0.

As an example, approval rules or overviews from DSL (Definitive Software Library) can be configured and used.
