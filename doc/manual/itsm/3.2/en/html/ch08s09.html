OTRS ITSM provides the following reports for the evaluation of key indicators in supervising Change Management. All reports can be accessed through the integrated report generator.

Number of Changes Within a Defined Period

The report allows the definition of a relative (eg. Changes within the last x days) or absolute time period (eg. Changes from date1 to date 2). In addition, it is also possible to define the Change status to be reported.

Reports are available in "CSV" or "Print" (PDF) output formats.
