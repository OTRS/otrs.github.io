Aktualisieren

Im OTRS Appliance Manager können Sie sehen ob Aktualisierungen für Ihr OTRS System zur Verfügung stehen.Sollte dies der Fall sein, wird eine entprechende Meldung sowie ein Update-Button angezeigt.Nach dem Klicken auf den Button aktualisiert sich das System vollständig automatisch.
