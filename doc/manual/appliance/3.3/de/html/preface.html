Vorwort

Das vorliegende Dokument beschreibt die Installation und Administration der OTRS Appliance. Die OTRS Appliance wird als ISO-Image (CD-Abbild) heruntergeladen und ermöglicht eine einfache und schnelle Installation eines OTRS-Systems inklusive aller dafür notwendigen Komponenten (Betriebssystem, Datenbank, Webserver, etc.)

Dieses Handbuch beschreibt die Installation und Administration der OTRS Appliance auf Basis der Virtualisierungs-Software "Oracle VM VirtualBox".Die Appliance kann auch mit Virtualisierung-Lösungen anderer Hersteller verwendet werden, oder auf nativer Hardware installiert werden. Dazu muss die heruntergeladene ISO-Datei auf eine CD-ROM gebrannt werden.
