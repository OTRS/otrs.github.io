Hauptmenü

Um die Systemeinstellungen der OTRS Appliance zu ändern, drücken Sie in der Informationsseite der Appliance auf F2.Geben Sie dann das bei der Installation vergebene Passwort ein um zum Hauptmenü zu gelangen.

Im Hauptmenü stehen Ihnen die folgenden 8 Einstellungsmöglichkeiten zur Verfügung:Abmelden, Netzwerk konfigurieren, Herunterfahren, Shell öffnen, System Passwort setzen, Zeitzone einstellen, Systemsprache ändern,Backup erstellen.
