Einloggen

Öffnen Sie nun einen Web-Browser und geben Sie die auf der Informationseite angezeigte IP-Adresse ein, gefolgt von /otrs/index.pl,also z.B. http://192.168.178.23/otrs/index.pl und geben sie als Benutzername "root@localhost" und als Passwort "root" ein.

Die Installation ist nun vollständig abgeschlossen und Sie können mit dem OTRS-System arbeiten.
