Storage

Click on the "Storage" icon and on the CD symbol close to "Controller: IDE". Then click on the small CD symbol in the "Attributes" section, and the select "Choose a virtual CD/DVD disk file...". Select the previously downloaded OTRS appliance ISO file and click on "Ok". The setup of the new virtual machine is now completed.
