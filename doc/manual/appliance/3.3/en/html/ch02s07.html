Network

The new virtual machine has been created now and can be found in the list of virtual machines in the left sidebar. Now click on the newly created machine, and then click on the "Settings" icon in the top menu bar. Then click on "Network" and then on "Adapter 1". Choose "Bridged Adapter" for "Attached to". This enables your OTRS system to be reached from the network for incoming connections.
