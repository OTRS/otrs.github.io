Log in

Now open a web browser and type in the IP address which was shown on the information screen after completing the installation, the IP address needs to be followed by /otrs/index.pl, e.g. http://192.168.178.23/otrs/index.pl. Use "root@localhost" as user name and "root" as password.

The installation is now completed, and you can work with your OTRS system now.
