Main menu

Open the system maintenance main menu by pressing the F2 key in the information screen. You need to enter your system password here.

In the main menu you have the following 8 possibilities: Logout, Configure network, Shutdown, Open Shell, Set system password, Configure time zone, Change system language, Create backup.
