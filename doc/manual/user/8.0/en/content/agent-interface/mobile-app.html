Mobile App¶

The OTRS mobile app offers you the possibility to use OTRS on a mobile device for accessing the agent interface. The app supports multiple accounts with one tap switching. Furthermore the app supports two-factor authentication methods.

The OTRS mobile app is available in Google Play and Apple App Store.

The usage of OTRS on a mobile device is the same as in the browser. This chapter only describes of how to link the app with an OTRS account.

Manage Accounts¶

After starting the app the account management screen is displayed. In this screen all linked OTRS distances are displayed.

Account Management Screen

To add an account:

  1. Tap the Add New button.
  2. Fill in the required fields.
  3. Tap the Save button.

Link Account Screen

The app now establishes connection with the web server. After successful connecting the app shows the dashboard of OTRS.

Note

It is possible to link multiple accounts. If multiple accounts are connected, the intro screen displays all connected accounts and the relevant OTRS account can be selected there.

To edit an account:

  1. Tap the Edit icon in the title bar.
  2. Tap the Edit button.
  3. Modify the fields.
  4. Tap the Save button.

Edit or Remove Account

To remove an account:

  1. Tap the Edit icon in the title bar.
  2. Tap the Remove button.
  3. Confirm the message by tapping OK button.

Using the App¶

After installing the app and linking a new account, the OTRS can be used like in a browser.

Dashboard in a Mobile Device

The app contains a title bar, where the name of the OTRS instance is displayed. The arrow icon at the top left corner switches back to the account management screen. The arrows at the top right corner can be used to switch forward and back within the views of OTRS.

The organizer sidebar is collapsed and can be expanded by using the toggle icon in the main menu bar.
