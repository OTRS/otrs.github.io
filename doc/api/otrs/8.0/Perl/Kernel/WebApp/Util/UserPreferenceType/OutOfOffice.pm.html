NAME

Kernel::WebApp::Util::UserPreferenceType::OutOfOffice - Wrapper for User OutOfOffice fields.

Fields()

The /Fields() method returns a form date range field according to the configuration.

FieldsValue()

The /FieldsValue() sets the current value of the field(s).

Set()

The /Set() method will be called to set the out of office dates.

GetError()

The /GetError() method will validate and output the correct error message when updating the date ranges.
