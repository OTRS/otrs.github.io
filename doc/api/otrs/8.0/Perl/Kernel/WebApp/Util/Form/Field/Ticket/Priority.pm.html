_GetPriorities

To get the priorities list. In case UserID is defined in the field, it just returns the priority list, otherwise it'll depend on the workflow (if configured).

    my $Priorities = $TicketUtil->_GetPriorities(
        TicketID => 123,
        UserID   => 123,
    );

    my $Priorities = $TicketUtil->_GetPriorities(
        TicketID       => 123,
        CustomerUserID => 'customer_user_id_123',
    );

    my $Priorities = $TicketUtil->_GetPriorities(
        QueueID => 123,
        UserID  => 123,
    );

Returns:

    $Priorities = {
        1 => 'Priority A',
        2 => 'Priority B',
        3 => 'Priority C',
    };
