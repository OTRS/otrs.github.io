ArticleFields()

Returns common article fields for a Short Message Service article.

    my %ArticleFields = $SMSObject->ArticleFields(
        TicketID  => 123,   # (required)
        ArticleID => 123,   # (required)
    );

Returns:

    %ArticleFields = (
        Sender => {                     # mandatory
            Label => 'Sender',
            Value => 'John Doe',
            Prio  => 100,
        },
        Subject => {                    # mandatory
            Label => 'Subject',
            Value => 'Article subject',
            Prio  => 200,
        },
        ...
    );

ArticlePreview()

Returns article preview for a Short Message Service article.

    my $ArticlePreview = $SMSObject->ArticlePreview(
        TicketID    => 123,     # (required)
        ArticleID   => 123,     # (required)
        ResultType  => 'plain', # (optional) plain|HTML. Default HTML.
        MaxLength   => 50,      # (optional) performs trimming (for plain result only)
    );

Returns article preview in scalar form:

    $ArticlePreview = 'My SMS message';

ArticleCustomerRecipientsGet()

Get customer users from an article to use as recipients.

    my @CustomerUserIDs = $SMSObject->ArticleCustomerRecipientsGet(
        TicketID  => 123,     # (required)
        ArticleID => 123,     # (required)
    );

Returns array of customer user IDs who should receive a message:

    @CustomerUserIDs = (
        'customer-1',
        'customer-2',
        ...
    );
