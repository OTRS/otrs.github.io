NAME

Kernel::System::DataValidation::Validator::KnowledgeBase::ArticleID - Validator that verifies a given ArticleID.

PUBLIC INTERFACE

Provides a Check method, that will be called by the Kernel::System::DataValidation backend.

Example()

Returns an example of a possible value to be verified.

Check()

Lookups the given ArticleID in the database. For more information about the faq backend, please refer to Kernel::System::FAQ.

    my $True = $Validator->Check(
        Arguments => [ { Valid => 1 } ],    # Defines if the lookup must be valid to success the check, default: 1
        Value     => 1,
    );

    my $False = $Validator->Check(
        Value => 'WrongArticleID',
    );
