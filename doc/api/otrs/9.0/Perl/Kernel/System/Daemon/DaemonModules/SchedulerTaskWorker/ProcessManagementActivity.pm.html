NAME

Kernel::System::Daemon::DaemonModules::SchedulerTaskWorker::ProcessManagementActivity - Scheduler daemon task handler module for ProcessManagement non-user Activities

DESCRIPTION

This task handler executes scheduler tasks delegated by ProcessManagement
